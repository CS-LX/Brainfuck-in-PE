# Brainfuck-in-PE

实验性项目：**第三种 DLL**——外壳是标准 Windows PE 动态链接库，代码节（`.text`）主体为明文 Brainfuck，由极小原生 Stub + VM 解释执行。

## 与常见 DLL 的区别

| 类型 | 代码节内容 | 执行方式 |
|------|------------|----------|
| 原生 DLL | x64/x86 机器码 | CPU 直接执行 |
| 托管 DLL | .NET IL | CLR |
| **本项目** | **明文 Brainfuck** | **VM 解释** |

## 实验约束

1. 产物为 `xxx.dll`，可被 `LoadLibrary` / 导入库正常加载  
2. **禁止欺骗**：不得将 BF 预编译/JIT 为机器码作为业务逻辑  
3. **禁止伪装**：不得把 BF 藏在 `.rsrc`、`.rdata` 等资源/数据节  
4. BF 明文必须位于 **`.text`（放指令的地方）**；用记事本打开 DLL 可搜索到 `+-<>[],.`
